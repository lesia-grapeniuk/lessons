Задача №2 За допомогою циклу FOR виведіть в консоль 10 рядків: Пункт №1 Пункт №2...
for (let i = 1; i <= 10; i++) {
console.log(`Пункт №${i}`);
}
В консолі також можна глянути результат
Цикл 'for' в JavaScript використовується для багаторазового виконання блоку коду з певною кількістю повторень. Синтаксис вже відтворений чуть вище. Давайте розглянемо кожну частину цього синтаксису:
1. Ініціалізація (Initialization): Цей вираз виконується один раз перед початком циклу. Зазвичай тут оголошується і задається початкове значення змінної-лічильника.
let i = 0 ;
2. Умова (Condition): Цей вираз перевіряється перед кожною ітерацією циклу. Якщо умова повертає 'true', виконується тіло циклу. Якщо 'false', цикл припиняється.
i < 10 ;
3. Оновлення (Update): Цей вираз виконується після кожної ітерації циклу. Зазвичай тут змінюється значення змінної-лічильника. Наприклад:
i++;
Отже, Порядок виконання:
1. Ініціалізація - виконується один раз перед початком циклу.
2. Умова - перевіряється перед кожною ітерацією.
3. Тіло циклу - виконується, якщо умова true.
4.Оновлення - виконується після тіла циклу.
5. Цей порядок повторюється, доки умова не стане false.
6. В консолі виводимо в зворотніх лапках (`інтерполяція`) текстове повідомлення, яке включає значення змінної 'i'.
7. console.log - це (як казав Женя, дебагер для бідних:)) функція, яка використовується для виводу інформації в консоль. Вона дозволяє розробникам виводити текстові повідомлення або значення змінних для діагностики та налагодження коду.
8. Пункт №${i} - це шаблонний рядок (template literal) в JavaScript. Шаблонні рядки створюються за допомогою зворотних апострофів (``) і дозволяють включати змінні та вирази всередину рядка.
9. 'Пункт №:' - це текстова частина рядка, яка буде виведена без змін.
10. '${i}' - це змінна або вираз JavaScript, значення якої буде вставлено на це місце в рядку. Наприклад, якщо i дорівнює 5, то ${i} буде замінено на 5.
11. Обчислення значення та виведення: коли цей рядок виконується, спочатку обчислюється значення змінної 'i', потім створюється новий рядок з підставленим значенням і цей рядок виводиться в консоль.

Задача №5 Створіть масив даних - 5 елементів, один з яких число 10. Обробіть масив за допомогою методу перебору. Перевіряйте елемент на відповідність числу 10, та у разі відповідності, виводьте в консоль Для вирішення цієї задачі потрібно створивши масив з пʼяти елементів - перебрати їх за допомогою циклу 'for'. Наступний крок - перевірити чи він строго дорівнює числу '10', та вивести в консоль, знайдене. Ось як він виглядає:

const array = [3, 10, 'hello', 42, true];

for (let i = 0; i < array.length; i++) {
if (array[i] === 10) {
console.log('Знайдено число:', array[i]);
}
}

В консолі ми отримаємо повідомлення: 'Знайдене число: 10'
Також можна вирішити це завдання такими методами масивів, як:
1. find
Метод 'find' повертає перший елемент масиву, який задовольняє умову вказану в функції зворотного виклику. Проте, якщо б у нас було декілька цифр '10', то знайшовши першу 'десятку' (що еквівалентне до 'true') - пошук припиниться, якщо не знайде жодного поверне 'undefined'
const array1 = [ 3, 10, 'hello js', 42, true ] ;
const found = array1. find ( element => element === 10 ) ;
if ( found !== undefined ) {
console. log ( 'Знайдено число 10:', found ) ;
}
2. some
Метод some схожий до попереднього, проте перевіряє, чи задовольняє хоча б один елемент масиву умову, вказану в функції зворотного виклику. Повертає булеве значення, тобто 'true'/'false'.
const array2 = [ 3, 10, 'hello', 42, true ] ;
const hasTen = array2. some ( element => element === 10 ) ;
if ( hasTen ) {
console. log ( 'Знайдено число 10', hasTen ) ;
}
3. filter
Метод filter створить новий масив з усіх елементів, які задовольняють умову, вказану в функції зворотного виклику.
const array3 = [ 3, 10, 10, 'привіт', 10, 42, false ] ;
const filtered = array3. filter ( element => element === 10 ; )
if ( filtered. length > 0 ) {
console. log ( 'Створено новий масив чисел 10:', filtered ) ;
}
4. forEach
Метод 'forEach' дозволить запустити функцію для кожного елемента масиву один раз. Функція не буде виконана для віддалених чи пропущених елементів масиву, але буде викликана для елементів які існують в масиві чи мають значення 'undefined'. Також хочу підкреслити, що не існує методу перервати роботу 'forEach' - виконаються всі цикли перебору! Цей метод найдоречніший на мою думку для вирішення цієї задачі. Маючи декільки одинакових значень які шукаються, в консолі ми побачимо кружечок з лівої сторони з відобреженням кількості знайдених значень. Метод масиву forEach - це те саме, що й цикл 'for', проте він зручніший тим, що коду набагато менше, а також він більш читабельний і зрозумілий. Використання 'forEach' допомагає уникнути помилок, пов'язаних з ініціалізацією ітератора, умовами завершення та кроками ітерації, що часто трапляються при написанні класичного циклу for. Автоматичне керування індексами: forEach автоматично обробляє індекси, тому немає потреби турбуватися про їх ініціалізацію, інкрементування та перевірку умов. Функціональний стиль: forEach сприяє використанню функціонального стилю програмування, де обробка кожного елемента масиву виконується за допомогою колбек-функції. Це робить код більш декларативним та виразним. Менша ймовірність помилок: Використання 'forEach' зменшує ймовірність помилок, пов'язаних з неправильним управлінням індексами або умовами циклу.
const array4 = [ 3, 10, 10, 'hello', 42, 10, true ] ;
array. forEach ( element => {
if ( element === 10 ) {
console. log ( 'Знайдено число 10:' , element ) ;
}
} )
